# Qutation

เว็บทำใบเสนอราคาแบบหน้าเดียว (HTML/CSS/JS) รองรับ:

- ล็อกอินผู้ใช้งานก่อนทำเอกสาร
- ฟอร์มใบเสนอราคา (ขายตรง QTC / ผ่านเอเจนต์ QTD)
- เพิ่ม/ลบลูกค้า และบันทึกข้อมูลใน LocalStorage
- เพิ่ม/ลบรายการสินค้า พร้อมราคาทุนและราคาขาย
- ส่วนลดแบบเปอร์เซ็นต์หรือบาท (ระดับรายการและระดับเอกสาร)
- เตือนราคาขายต่ำกว่าทุน และบังคับอนุมัติก่อนบันทึก
- ใส่ชื่อผู้จัดทำ ชื่อเซลล์ ผู้อนุมัติ และสถานะส่งอนุมัติ
- พิมพ์หรือบันทึก PDF (ซ่อนราคาทุนตอนพิมพ์)
- หน้าแดชบอร์ดสรุปรายวัน/รายเดือน/ทั้งหมด

## วิธีใช้งาน

เปิดไฟล์ `/tmp/workspace/AdminHomeSmile/Qutation/index.html` ผ่านเว็บเซิร์ฟเวอร์ เช่น:

```bash
cd /tmp/workspace/AdminHomeSmile/Qutation
python -m http.server 4173
```

แล้วเปิด `http://127.0.0.1:4173/index.html`
