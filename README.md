# Qutation

ระบบจัดการใบเสนอราคาและสต็อกสินค้า แบบ HTML/CSS/JS ล้วน ไม่ต้องติดตั้งอะไรเพิ่ม

ระบบถูกแยกออกเป็น **2 ระบบอิสระ** ที่ใช้งานได้แยกกัน:

---

## 📄 ระบบใบเสนอราคา (`quotation.html`)

ฟีเจอร์หลัก:
- ล็อกอินผู้ใช้งานก่อนทำเอกสาร
- ฟอร์มใบเสนอราคา (ขายตรง QTC / ผ่านเอเจนต์ QTD)
- เพิ่ม/ลบลูกค้า และบันทึกข้อมูลใน LocalStorage
- เพิ่ม/ลบรายการสินค้า พร้อมราคาทุนและราคาขาย
- ส่วนลดแบบเปอร์เซ็นต์หรือบาท (ระดับรายการและระดับเอกสาร)
- เตือนราคาขายต่ำกว่าทุน และบังคับอนุมัติก่อนบันทึก
- ใส่ชื่อผู้จัดทำ ชื่อเซลล์ ผู้อนุมัติ และสถานะส่งอนุมัติ
- พิมพ์หรือบันทึก PDF (ซ่อนราคาทุนตอนพิมพ์)
- แดชบอร์ดสรุปรายวัน/รายเดือน/ทั้งหมด
- ประวัติใบเสนอราคาทั้งหมด

---

## 📦 ระบบสต็อกสินค้า (`stock.html`)

ฟีเจอร์หลัก:
- ล็อกอินผู้ใช้งานก่อนใช้งาน
- สรุปสต็อกคงเหลือของแต่ละสินค้า
- เพิ่มสต็อก (รองรับเพิ่มสินค้าใหม่ได้)
- เบิก/หักสต็อก โดยต้องกรอก: ชื่อผู้เบิก, ชื่อลูกค้า, วัตถุประสงค์
- รายงานธุรกรรมสต็อกย้อนหลัง
- ระบบจะไม่อนุญาตให้เบิกเกินจำนวนคงเหลือ

---

## วิธีใช้งาน

รันเว็บเซิร์ฟเวอร์จากโฟลเดอร์โปรเจกต์:

```bash
cd /path/to/Qutation
python -m http.server 4173
```

จากนั้นเปิดในเบราว์เซอร์:

| หน้า | URL |
|------|-----|
| หน้าหลัก (เลือกระบบ) | `http://127.0.0.1:4173/index.html` |
| ระบบใบเสนอราคา | `http://127.0.0.1:4173/quotation.html` |
| ระบบสต็อกสินค้า | `http://127.0.0.1:4173/stock.html` |

แต่ละระบบมีหน้าล็อกอินของตัวเอง และทำงานได้อิสระโดยไม่พึ่งพากัน

---

## โครงสร้างไฟล์

```
index.html        — หน้าหลัก (เลือกเข้าระบบ)
quotation.html    — UI ระบบใบเสนอราคา
quotation.js      — Logic ระบบใบเสนอราคา
stock.html        — UI ระบบสต็อกสินค้า
stock.js          — Logic ระบบสต็อกสินค้า
common.js         — Utilities ร่วม (loadJSON, formatMoney ฯลฯ)
styles.css        — สไตล์ชีตร่วม
```

ข้อมูลทั้งหมดเก็บใน **LocalStorage** ของเบราว์เซอร์ แยก key ระหว่างใบเสนอราคาและสต็อก
